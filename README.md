## About The Project

The first solo project in Scrimba's Learn React course. This project will apply everything that has been learned in the first part of the course "Static Pages"

Requirements are as follows :
1. Build it from scratch
2. Fill in with my own information
3. Seperate components for :
    * Info
    * About
    * Interests
    * Footer

## Getting Started
To get a local copy up and running follow these simple steps.

### Prerequisites
To run this project, you need to install npm.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/ImLostPlsHelp/Scrimba---Digital-Business-Card
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the server
    ```
    npm run start-dev
    ```
